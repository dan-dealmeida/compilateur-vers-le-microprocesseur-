# C Compiler with Lex/Yacc

A basic C compiler developed using Lex (lexical analyzer) and Yacc (syntax analyzer).

## Features
- Lexical analysis using `lex`.
- Syntax analysis using `yacc`.
- Basic assembly-like code generation (`AFC`, `STORE`, `ADD`, `MUL`, etc.).

## How to use
1. **Compile the compiler:**
   ```bash
   make
   ```
2. **Run the test program:**
   ```bash
   make test
   ```
3. **Clean generated files:**
   ```bash
   make clean
   ```
